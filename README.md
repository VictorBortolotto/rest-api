- [MySQL-Community-Server](https://dev.mysql.com/downloads/)

### :cd: Rodando a Aplicação

```bash 

# Clone o repositório

$ git clone <https://github.com/VictorBortolotto/rest-api>

# Acesse a pasta

cd rest-api/restapi/demo

# Rode o comando 

mvn spring-boot:run

# Acesso a porta http://localhost:8080/users

#Faça as requisições usando POSTMAN, Talend API Tester - Free Edition (Extensão Google-Chrome) ou Rest client (VScode) com os comandos: 

# POST para adicionar usuários
# PUT para atualizar os usuários
# GET para acessar o usuário
# DELETE para deletar o usuário

#Exemplo GET para acessar usuário 1
#http://loclahost:8080/users/1 

```
